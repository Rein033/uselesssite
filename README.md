# uselesssite
uselesssite.nl is precies wat de naam zegt: compleet nutteloos. Klik op de knop en er gebeurt iets willekeurigs — van chaos en memes tot rare geluiden en onverwachte pagina’s. Geen doel, geen nut, geen uitleg. Gewoon klikken, ontdekken en tijd verspillen op de meest zinloze manier mogelijk.
